[GEMMA_ROUTER]
규칙 (우선순위 순):

1. 봇 이름이 직접 호칭된 경우 → 호칭된 봇만 포함합니다.
   - 호칭된 봇이 다른 봇에게 무언가를 지시하도록 요청받은 경우, 지시를 받은 봇(호칭된 봇)만 포함하고 지시 대상 봇은 포함하지 않습니다.
   - 예: "찌몽아 아루한테 이거 시켜" → [찌몽] (아루는 찌몽이 알아서 처리할 대상)
   - 예: "찌몽아 안녕" → [찌몽]
   - 업무와 무관한 잡담, 게임, 농담도 동일하게 적용합니다.

2. 여러 봇이 각각 독립적으로 호칭된 경우 → 호칭된 봇 모두 포함합니다.
   - 예: "찌몽아 안녕, 아루야 안녕" → [찌몽, 아루]

3. 특정 봇 이름 없이 역할 관련 요청인 경우 → 해당 역할의 봇을 선택합니다.
   - 예: "코드 리뷰해줘" → [아루(reviewer)]
   - 예: "개발해줘" → [개발이(developer)]

4. 모든 봇에게 해당하는 인사, 공지, 출석 체크 등 → 전체를 선택합니다.
   - 예: "다들 안녕?", "출석 체크"

5. 봇 이름/역할 언급이 전혀 없고 봇과 완전히 무관한 내용 → 빈 배열을 반환합니다.
