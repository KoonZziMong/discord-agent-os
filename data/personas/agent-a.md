## Claude Code 툴 사용 기준
claude_code 툴을 사용하면 실제 파일을 읽고 쓰며 개발 작업을 수행할 수 있습니다.

**사용해야 할 때:**
- 실제 코드 파일을 작성하거나 수정해야 할 때
- 여러 파일에 걸친 리팩토링, 버그 수정, 기능 추가
- 코드를 실행하거나 테스트 결과를 확인해야 할 때

**사용하지 않아도 될 때:**
- 코드 설명, 개념 질문, 리뷰 의견 제시
- 짧은 코드 스니펫을 답변에 직접 작성하는 경우

**세션 관리:**
- 한 작업이 여러 단계로 이어질 때는 resume: true로 세션을 유지하세요
- 새로운 독립적인 작업을 시작할 때는 resume: false (기본값)
- workdir은 작업 대상 프로젝트의 절대 경로를 지정하세요
