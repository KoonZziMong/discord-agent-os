# discord-ai-team

Discord 멀티 에이전트 AI 봇 시스템. 3개의 봇(찌몽/아루/센세)이 각자 독립된 채널에서 운영됩니다.

## 슬래시 커맨드

### `/status`

봇의 현재 운영 상태를 조회합니다.

**권한**: 관리자(Administrator)만 사용 가능  
**가시성**: 명령어를 실행한 본인에게만 표시됨 (ephemeral)

#### 출력 필드

| 필드 | 설명 |
|------|------|
| 업타임 | 봇 프로세스가 마지막으로 시작된 이후 경과 시간 (예: `2일 3시간 15분 42초`) |
| 핑 | Discord WebSocket 레이턴시 (밀리초) |
| 서버 수 | 봇이 현재 참여 중인 Discord 서버(길드) 수 |
| 메모리 사용량 | 현재 힙 메모리 사용량 (MB 단위) |
| Node.js 버전 | 봇이 실행 중인 Node.js 런타임 버전 |

#### 예시 출력

```
┌─────────────────────────────┐
│          봇 상태             │
├──────────┬──────────┬───────┤
│ 업타임   │ 핑       │ 서버 수│
│ 1일 4시간│ 48ms     │ 3개   │
├──────────┴──────────┴───────┤
│ 메모리 사용량 │ Node.js 버전 │
│ 124.3 MB     │ v22.13.1     │
└─────────────────────────────┘
```

> **참고**: 실제 출력은 Discord 임베드 형식으로 표시됩니다. 스크린샷은 추후 추가 예정입니다.

#### 오류 처리

- 관리자 권한이 없는 경우: `이 커맨드는 관리자만 사용할 수 있습니다.` (본인에게만 표시)
- 조회 중 오류 발생 시: `상태 조회 중 오류가 발생했습니다.` (본인에게만 표시)
